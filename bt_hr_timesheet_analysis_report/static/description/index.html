Timesheet Analysis Report

This module from BroadTech IT Solutions introduces an xls report for the timesheet analysis of Users during a selected period .

The report can be generated for a selected user or the entire users. Unless specified the user it would return the whole time sheet from the system between the selected periods.

The wizard action return the xls file to downoad.
